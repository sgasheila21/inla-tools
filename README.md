## Installation
1. Create the '.env' file  
   Copy the example environment file to create your local '.env' configuration file:

   ```bash
   cp .env.example .env
2. Install dependencies

   ```bash
   composer install
3. Generate app key

   ```bash
   php artisan key:generate
4. Run migrations

   ```bash
   php artisan migrate
5. Seed database

   ```bash
   php artisan db:seed
6. Start server
   ```bash
   php artisan serve
